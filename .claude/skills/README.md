# Skills

This directory contains the skill library from this repo.

## Usage

Pick the skill folders you need and copy them into your project's `.claude/skills/` directory.

## Install

### Plugin install

```bash
/plugin marketplace add owner/repo
/plugin install flox@flox
```

### Manual install

```bash
cp -R skills/<skill-name> /path/to/your-project/.claude/skills/
```
